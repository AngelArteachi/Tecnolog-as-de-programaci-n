# Aplicación Tarea 3 - Parcial 2

Este es el código fuente de la aplicación solicitada. Es una aplicación muy sencilla creada con Node.js usando el framework Express que retorna una página web estática básica (un "básico de front"). 

Esta aplicación ha sido diseñada para poder ser desplegada fácilmente tanto en servicios donde el servidor corre (App Services) como en servicios exclusivos de estáticos (Static Web Apps).
